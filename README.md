# 时间管理小助手

智能语音时间管理助手，支持语音添加任务和提醒，可安装在手机上像原生 App 一样使用。

## 功能特点

- 🎤 **语音输入**：按住说话，自动识别任务和提醒
- 📋 **任务清单**：优先级管理，完成状态追踪
- ⏰ **智能提醒**：精确到分钟的提醒，支持浏览器通知
- ⏱️ **快速计时**：番茄钟倒计时
- 📱 **PWA 支持**：可安装到手机主屏幕，离线使用

## 使用方法

### 本地使用（电脑）

直接用浏览器打开 `index.html` 文件即可。

### 手机使用（推荐）

手机无法直接打开本地文件，需要部署到网上：

#### 方案一：GitHub Pages（免费，推荐）

1. 注册 [GitHub](https://github.com) 账号
2. 创建新仓库，名字为 `yourname.github.io`
3. 上传这三个文件到仓库
4. 访问 `https://yourname.github.io`

#### 方案二：Vercel（免费）

1. 注册 [Vercel](https://vercel.com) 账号
2. 导入 GitHub 仓库或拖拽上传文件
3. 自动获得网址，访问即可

#### 方案三：Netlify（免费）

1. 访问 [Netlify Drop](https://app.netlify.com/drop)
2. 直接拖拽文件夹上传
3. 立即获得网址

### 安装到手机主屏幕

**iPhone (Safari)：**
1. 用 Safari 打开网址
2. 点击底部"分享"按钮
3. 选择"添加到主屏幕"
4. 点击添加

**Android (Chrome)：**
1. 用 Chrome 打开网址
2. 点击菜单 → "添加到主屏幕"
3. 或在弹出的提示中点击"安装"

## 语音指令示例

| 语音指令 | 效果 |
|---------|------|
| "下午三点开会" | 创建下午3点的提醒 |
| "20分钟后喝水" | 创建20分钟后的提醒 |
| "买牛奶高优先级" | 创建高优先级任务 |
| "明天上午十点交报告" | 创建明天10点的任务 |
| "今晚八点看电影" | 创建今晚8点的提醒 |

## 技术说明

- 纯前端实现，无需后端
- 数据存储在浏览器本地（localStorage）
- 使用 Web Speech API 进行语音识别
- 支持离线使用（PWA）

## 浏览器兼容性

- Chrome / Edge：完全支持
- Safari：支持（语音识别需 iOS 14.5+）
- Firefox：支持（语音识别需开启实验性功能）
